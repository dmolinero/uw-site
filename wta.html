Data Visualization Tool

Washington Trails Association

Our Opportunity

The Problem

The Washington Trails Association had accumulated a large amount of donor data, but had no established method for visualizing it.

Contraints

The WTA had recently purchased a tableau subscription and wanted to use Tableau for the tool. We also only had about four weeks to complete the project.

Client's Goals

They were aware of some general trends among their donors, but wanted to be able to visual donations goegraphically and look for trends over time.

The Existing Data

The existing data they had was in csv format. There was one large file witho about 100,000 rows of data spanning a 10 year time range.

Our Solution

A Tool with Multiple Visualizations

Through user studies, our team researchers discovered that the WTA staff would need several different visualizations in order to obtain the information we were looking for. With that in mind, we made the landing screen for our tool consist of a menu which funneled users to the visualization that would meet their needs.

Easy for Anyone

Although we wanted to use Tableau, we discovered that not all users would have enough technical skill to use a dashboard inside Tableau. Because of this, we built the individual visualizations inside Tableau, then embedded them in a web app. The web app allowed users to interact with the dashboard with more familiar controls.
